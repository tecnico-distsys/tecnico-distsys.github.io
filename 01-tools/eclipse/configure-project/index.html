Como configurar um projecto no Eclipse

Para começar, os ficheiros do projectos deverão ser colocados numa pasta de trabalho.

Se existirem os ficheiros de configuração do Eclipse (.project, .classpath):

  • 'File', 'Import...', 'General'-'Existing Projects into Workspace'
  • 'Select root directory' e 'Browse' até à pasta do projecto.
  • Confirmar que está tudo como desejado e 'Finish'.

Se não existirem os ficheiros de configuração do Eclipse:

  • Criar um 'New Project', do tipo 'Java Project'.
  • Remover a seleção 'Use default location' e 'Browse' até à pasta raiz do projecto (pasta mãe das 'sources').
  • Preencher o 'Project name'.
  • Nota: apesar de existir a opção Import Java Project from Existing Ant Buildfile, desaconselhamos o seu uso pois frequentemente resulta em projectos configurados de forma imperfeita.

Para executar o programa:

  • Seleccionar a classe principal (que tem o método main()
  • Carregar no botão 'Run' (o botão verde de 'play'.
  • Indicar argumentos do programa usando 'Run Configurations' (drop-down).

Para depurar (debug) o programa:

  • Seleccionar novamente a classe principal.
  • Carregar no botão 'Debug' (o botão com o insecto).
  • Nota: no caso do programa ser executado através do Ant, as instruções para debug são diferentes.

Para usar o Ant dentro do Eclipse:

  • Abrir a Ant View (Window > Show View > Ant).
  • Arraste o build.xml do seu projecto para a Ant view. Os alvos (targets) do build.xml arrastado serão apresentados.
  • Clique 2x sobre o alvo pretendido para o lançar.
